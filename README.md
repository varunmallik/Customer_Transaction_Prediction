# Customer Transaction Prediction

## Overview

This project predicts whether a customer will make a future transaction using machine learning classification algorithms.

## Problem Statement

Develop a binary classification model to predict customer transactions using anonymized numerical features.

## Dataset

Customer Transaction Prediction Dataset
